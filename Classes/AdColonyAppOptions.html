Overview

AdColonyAppOptions objects are used to set configurable aspects of SDK state and behavior, such as a custom user identifier. The common usage scenario is to instantiate and configure one of these objects and then pass it to [AdColony configureWithAppID:options:completion:]. You can use one of the constants below to configure a specific option. Note that you can also reset the current options object the SDK is using by passing an updated object to setAppOptions:.
